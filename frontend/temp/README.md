# Templates e referências

Esta pasta guarda apenas textos e fragmentos que serão migrados manualmente do Canvas do Gemini.

- Nenhum arquivo aqui é código de produção.
- Os conteúdos servem como referência temporária para a IA.
- Sinta-se à vontade para colar rascunhos, fluxos e padrões visuais antes de levá-los para `src/`.
